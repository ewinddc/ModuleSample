# ModuleSample
a sample for module development on android platform

## 介绍
为了在项目中推进android组件化技术，写了这个示例。示例的模块架构很简单，一个app主模块，2个业务模块module_a和module_b，一个common库模块，基本上可以勾勒出一个常用的组件化模型。

此项目提供了一些组件化过程中需要注意的问题解决方案以及一些功能特性的展示
- 模块解耦
- 模块间通信机制
- 模块间资源共享和冲突问题
- 模块初始化方式
